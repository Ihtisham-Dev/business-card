# 💼 Business Card

A simple **Business Card** webpage built using **HTML5** and **CSS3** as part of my Frontend Development learning journey.

---

## 📖 About the Project

The **Business Card** project is a beginner-friendly webpage that displays personal and professional information in a clean and organized layout. It includes a profile image, name, designation, company, contact details, portfolio link, and social media links.

This project was created while learning the fundamentals of **CSS** and applying them to build a real webpage from scratch.

---

## ✨ Features

* Professional business card layout
* Profile image
* Name, designation, and company information
* Contact details (Email & Phone)
* Portfolio link
* Social media links
* Clean and centered design
* Beginner-friendly and semantic HTML structure

---
